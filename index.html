XML

En HTML se pueden incorporar animaciones usando en conjunto CSS y Javascript. Hay que recordar utilizar un elemento DIV como contenedor, fijando la posición en relativa.La animación irá en un elemento DIV de animación, fijado en posición absoluta. En Javascript se coloca la programación de la animación, que en su norma general utilizan temporizadores para los cambios graduales de los elementos a animar.

  • Cuando carga una página web
  • Cuando se carga una imagen
  • Cuando se pulsa una tecla
  • Cuando se hace clic en un elemento

TABLA XML VS JSON

Se refiera a la utilización del metodo "addEventListener()" para controlar eventos de un elemento específico de forma ordenada y evitando sobreescribir eventos. Estos trabajan con cualquier elemento DOM de la ventana. En el ejemplo siguiente podemos ver como iteractuan 2 event listeners. 

  <!doctype html>
    <html lang="en-US">
    
    <head>
    <meta charset="utf-8">
    <title>Web Animations API implicit keyframes</title>



    </head>
    
    <body>

<button id="example-button">You have not clicked this button.</button>
<button id="reset-button">Click this button to reset the first button.</button>



    
    <script>
    const buttonToBeClicked = document.getElementById("example-button");

const resetButton = document.getElementById("reset-button");

// the text that the button is initialized with
const initialText = buttonToBeClicked.textContent;

// the text that the button contains after being clicked
const clickedText = "You have clicked this button.";

// we hoist the event listener callback function
// to prevent having duplicate listeners attached
function eventListener() {
  buttonToBeClicked.textContent = clickedText;
}

function addListener() {
  buttonToBeClicked.addEventListener(
    "click",
    eventListener,
    {
      passive: true,
      once: true
    }
  );
}

// when the reset button is clicked, the example button is reset,
// and allowed to have its state updated again
resetButton.addEventListener(
  "click",
  () => {
    buttonToBeClicked.textContent = initialText;
    addListener();
  }
);

addListener();
    </script>
    </body>
    
    </html>
